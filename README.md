# 📊 Especialización en SQL: De la Teoría a la Aplicación Industrial

Este repositorio documenta mi proceso de formación técnica en SQL, utilizando como eje central el programa de **[aprendersql.es](https://aprendersql.es)** y **https://sqlbolt.com/lesson/select_queries_introduction**. 
,también vi sql en python inicial UTN(SQLite), y en humai(próximamente)
El objetivo es consolidar el dominio de bases de datos relacionales y aplicar dicho conocimiento en escenarios de negocio reales.

---

## 🛠️ Metodología de Trabajo
Mi aprendizaje se estructura en tres etapas progresivas por cada módulo:
1. **Fundamentación Técnica:** Implementación de scripts basados en la teoría del curso.
2. **Práctica Dirigida:** Resolución de ejercicios de sintaxis y lógica de consultas, en cada area por ejemplo SELECT/FROM van a ver ejercicios aplicados a un dataset real sacado de kaggle
3. **Casos de Industria:** Al finalizar cada nivel, desarrollo un proyecto práctico aplicado a sectores reales (Retail, Finanzas, Logística, etc.) para demostrar la resolución de problemas
   de negocio.Estos proyectos prácticos van a ser integrales.

## 📂 Contenido del Repositorio
*   **`SQL/01-Basico/`**: Creación de tabla con `CREATE TABLE`, `INSERT INTO`.Uso de `SELECT`, `WHERE`, `ORDER BY`, `LIMIT` y operadores lógicos.
*   **`SQL/02-Intermedio/`**: Consultas multi-tabla (`JOINs`), Funciones de Agregación y `GROUP BY`.
*   **`SQL/03-Avanzado/`**: Subconsultas, CTEs (Common Table Expressions) y Funciones de Ventana.
*   **`Casos-Industria/`**: Proyectos finales de módulo con enfoque en análisis de datos empresarial(básico, intermedio y avanzado; de forma progresiva)
